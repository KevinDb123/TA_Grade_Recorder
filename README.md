# 中科大学生名单作业/实验成绩登记工具

一个用于助教登记作业、实验和平时成绩的本地工具，适配中科大学生名单类 Excel。

它的目标是直接在现有 `xlsx` 名单上完成录分、批量录分和成绩查看，而不是要求用户重新整理表格。

## 桌面版获取方式

桌面版请直接到 GitHub 的 `Releases` 页面下载：

- 进入仓库的 `Releases`
- 下载最新版本里的 Windows 安装包
- 安装后直接运行即可

## 本地源码运行

如果你是开发者，想从源码运行：

```bash
pip install -r requirements.txt
python app.py
```

然后访问：

```text
http://127.0.0.1:5000
```

## 技术栈

- Flask
- openpyxl
- HTML / CSS / JavaScript

## 功能概览

- 读取当前目录下的 `.xlsx` 文件
- 导入新的 `.xlsx` 文件
- 自动识别“平时成绩”列区间
- 支持不同数量的作业、实验或次数列
- 支持新增、删除、重命名成绩列
- 支持单人录分
- 支持批量录分
- 支持批量录分预览
- 支持“覆盖已有成绩”或“跳过已有成绩”
- 支持成绩查看和搜索
- 成绩查看页固定每页显示 5 人

## 适用表格结构

当前版本适合这类 Excel：

- 第 6 行包含“平时成绩”总表头
- 第 7 行是各次作业、实验或平时成绩子表头
- 第 8 行开始是学生数据
- 学号在第 2 列
- 姓名在第 3 列
- “平时成绩”下面的列数可以变化

## 使用方式

### 录分页面

你可以在录分页面完成：

- 切换 Excel 文件
- 导入新文件
- 选择当前次数列
- 重命名当前列
- 新增一列
- 删除当前列
- 搜索学生
- 保存单人成绩
- 批量保存成绩

### 批量录分格式

格式一：每行一个“学号 成绩”

```text
21007001 95
21007002 88
21007003 91
```

格式二：只输入学号，再统一填写成绩

```text
21007001
21007002
21007003
```

然后在“统一成绩”里输入同一个分数即可。

### 成绩查看页

你可以在成绩查看页：

- 查看某一次成绩
- 搜索学生成绩
- 查看已评分人数
- 查看未评分人数

## 注意事项

- 写入 Excel 前请关闭正在占用文件的 Excel 或 WPS
- 当前工具面向本地使用，不包含多人协作和权限系统
- 建议先保留一份原始名单副本
